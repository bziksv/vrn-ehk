# Модуль "Ammina: Отправка почты через SMTP и DKIM подпись писем" для 1с-битрикс
Модуль "___Ammina: Отправка почты через SMTP и DKIM подпись писем___" позволяет:

* Отправлять почту сайта через сторонние SMTP серверы.
* Подписывать письма DKIM подписью.

## Подробная документация на модуль
Расположена по адресу: [Просмотреть документацию](https://www.ammina.ru/documentation/ammina.smtp/)

## Получение модуля
Для получения модуля необходимо перейти по адресу [Переход на сайт 1С-Битрикс.Маркетплейс](https://www.ammina.ru/bitrix/redirect.php?goto=https%3A%2F%2Fmarketplace.1c-bitrix.ru%2Fsolutions%2Fammina.templates%2F&event1=marketplace&event2=ammina.templates), нажать кнопку "Установить" и указать адрес своего сайта. 

## Использование модуля

* Создайте почтовый домен (один или несколько), с которых будет отправляться почта.
* При необходимости сгенерируйте DKIM ключи (либо вставьте имеющиеся).
* Добавьте DNS записи
* Создайте почтовые аккаунты SMTP для доменов (Если отправка почты будет через SMTP) 
* В настройках модуля поставьте галочку "Активировать модуль"
